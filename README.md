# ComfyUI ShellLink 用户使用手册

ShellLink 是一个为 ComfyUI 打造的沉浸式、卡片化任务管理面板。它允许你将复杂的节点参数提取到直观的界面中，通过“输入”和“输出”模块的组合，快速构建、管理并批量运行任务。

## 1. 顶部全局工具栏

位于面板最上方，负责全局任务与工作流的管理操作。

- **[+ 新建任务]**：在当前面板的最末尾，添加一个空白的“任务卡片”。
- **[+ 新建模块]**：在当前选中的任务卡片内部，添加一个新的参数模块（默认为输入类型）。

### 1.1 导入 (向下的箭头图标)

支持读取 JSON 格式的参数配置文件，快速转换为任务卡片。菜单分为“从剪切板导入”和“从本地文件导入”两组，每组均包含以下三种导入模式：

- **创建任务**：读取 JSON 数据，将每个对象作为一个全新的任务卡片创建并添加到面板末尾。JSON 中的每个键值对将自动转换为卡片内对应的参数模块。
- **追加模块**：智能识别当前选中的任务位置（或面板末尾），将 JSON 数据中的键值对作为新模块，**追加到现有的任务卡片内部**。如果导入的数据数组长于现有的卡片数量，系统会自动创建新卡片来容纳多余的数据。
- **追加模块到选中**：仅对当前**已选中**的卡片（支持 Ctrl/Shift 多选）进行操作。导入的数据将按顺序严格追加到你所选定的那些卡片中。

### 1.2 运行与执行

- **▶ 运行**：仅执行当前选中的任务卡片，或者选中的特定输出模块。
- **⏭️ 运行全部 (下拉选项)**：一键将面板中所有的任务卡片排队发送给 ComfyUI 后端。系统会智能利用缓存机制，依次接力执行，并在卡片标题下方显示绿色的实时进度条。

### 1.3 导出 (向右的箭头图标)

用于备份配置或整理生成的图像文件。

- **导出JSON**
    - **导出内容切换 (输入 / 输出 / 全部)**：在菜单顶部的切换开关，决定导出的 JSON 文件中是仅包含输入参数、仅包含生成的图片路径，还是两者都包含。
    - **下载到本地**：将生成的 JSON 配置直接保存到你的电脑中，作为工作流预设文件备份。
    - **复制到剪切板**：将生成的 JSON 文本直接复制到剪切板，方便快速分享给他人或粘贴到外部文本编辑器。
- **整理输出 (文件管理)**
    - **移动到子文件夹**：将当前面板上生成的图片，从默认的 output 目录**物理移动**到以当前工作流命名的子文件夹内（例如 `output/ShellLink/你的工作流名/`），同时自动按“任务名_模块名.png”的规则对文件进行重命名。面板上的图片显示路径会无缝刷新，防止图片裂开。
    - **复制到子文件夹**：与“移动”功能类似，但仅**复制**文件到子文件夹并重命名，原有的输出文件仍保留在原处。

## 2. 任务卡片操作

卡片是任务的基本单元，代表了一次独立的 ComfyUI 运行指令。

- **重命名**：点击卡片标题栏默认的 `##1` 名称，即可自定义任务名称（如：赛博朋克风格测试）。
- **任务进度条**：卡片标题下方的分割线。在运行当前任务时，它会变成绿色的进度条实时伸展；如果底层节点运行报错，它会变成醒目的红色呼吸灯进行警示。
- **拖拽排序**：按住卡片的标题栏左右拖动，可以自由跨越调整任务的执行顺序。
- **多选操作**：按住 `Ctrl` (或 `Cmd`) 点击卡片空白处可进行多选；按住 `Shift` 可进行首尾连选。
- **删除**：点击卡片右上角的 `✖` 按钮可删除该任务。如果多选了多个任务，点击任意一个 `✖` 即可实现批量删除。

## 3. 居中动态工具栏

当你选中一个“任务卡片”或卡片内的“具体模块”时，顶部工具栏中间会浮现**动态工具栏**。它包含通用操作以及针对特定类型模块的专属设置。

### 3.1 通用神器：克隆与格式刷 (右侧区域)

这两项极其强大的功能**既适用于单个模块，也适用于整个任务卡片**，系统会根据你当前的选中目标智能生效：

- **[克隆] (矩形重叠图标)**：
    - **克隆模块**：选中模块后点击，会原样复制该模块，并紧接着在下方插入。
    - **克隆卡片**：选中整个任务卡片后点击，会原样复制整个任务卡片（包含内部所有模块的配置），并在右侧相邻位置插入一张全新的卡片。
- **[格式刷] (刷子图标)**：点击开启格式刷模式（图标变为橙色），右键点击或按 `ESC` 键可随时退出。
    - **模块级格式刷**：选中模块开启格式刷后。点击其它模块，会将其参数配置直接替换为源模块的配置；**点击任意卡片内部的空白区域，会直接粘贴插入该模块。**
    - **卡片级格式刷**：选中整个卡片开启格式刷后。点击其它任务卡片，会用源卡片的内部模块布局**完全覆盖替换**目标卡片；**点击面板背景的卡片间隙处（空白大区域），会直接在该位置粘贴生成一张完整的克隆卡片。**

### 3.2 跨卡片批量操作 (方块叠加图标)

专为处理拥有大量任务的复杂工作流设计。

- **[同步] 同步参数**：将当前模块的所有设置（关联节点、控件、宽高等），一键同步覆盖给面板中所有**同名（或同序号）**的同类模块。注意：它只同步配置参数，绝不改变目标模块的输入/输出属性。
- **[同步] 删除相同模块**：危险操作。一键删除所有任务卡片中，与当前选中模块**同名（或同序号）**的所有模块。
- **[批量] 选择相同模块**：一键高亮并选中所有任务卡片中的同名模块，方便随后利用工具栏进行统一的参数修改。
- **[批量] 批量向后移动**：将选中的模块（支持多选）统一向后（向右）移动一个任务卡片的位置。例如：任务 1 的模块去到任务 2，任务 2 的去到任务 3。末尾溢出时会自动新建空白卡片。
- **[批量] 批量向前移动**：将选中的模块统一向前（向左）移动一个任务卡片的位置。最前方任务 1 中的模块将保持在原地。

### 3.3 输入模块 (Input) 专用选项

用于向 ComfyUI 节点发送参数（如提示词、数值、布尔开关、图片/视频上传）。

- **[重置] (逆时针箭头)**：一键清空当前模块的关联节点和填写的参数值。
- **[拾取节点] (准星图标)**：点击后界面进入沉浸式暗化模式，此时去 ComfyUI 原生画布上点击任意一个节点，即可自动抓取该节点 ID 并完成绑定。
- **[关联节点] (下拉框)**：手动从工作流具有层级的节点树（Group）中选择要控制的节点。
- **[绑定参数] (下拉框)**：选择要控制目标节点上的具体参数（如 `text`, `seed`, `image` 等）。选择后，模块下方的输入框会自动适配该参数的类型：
    - **文本/数值**：显示为输入框。
    - **布尔值**：显示为开关按钮。
    - **下拉选项**：显示为专属的下拉菜单。
    - **媒体上传**：当绑定的是图像、视频或音频上传节点时，模块会自动变为**上传拖拽区**。点击或将本地文件拖入该区域，系统会自动将其加密上传至服务器并完成绑定。
- **[参数框高度适配]**：勾选后，长文本输入框会根据文本内容的多少自动撑开高度；取消勾选则保持固定高度并出现滚动条。

### 3.4 输出模块 (Preview) 专用选项

用于接收和显示 ComfyUI 节点生成的图像、视频或音频结果。

- **[管理生成记录] (网格图标)**：点击进入或退出“网格视图”模式。在网格视图下，你可以直观地管理该模块生成的所有历史媒体记录（详情见第 4 节）。
- **[重置] (逆时针箭头)**：一键清空当前模块的关联节点和展示设置。
- **[拾取节点] / [关联节点]**：与输入模块功能一致，用于绑定诸如 `SaveImage` 或 `SaveVideo` 等输出节点。
- **[比例预设] 与 [W : H]**：设置图像/视频在卡片中的展示比例（如 `16:9`, `1:1`），或手动输入宽高比。
- **[显示模式]**：
    - **显示全部 (Contain)**：保证完整看到生成图片的全貌，如果比例不符四周会留白。
    - **填充 (Cover)**：图片放大撑满整个模块框，超出模块比例的部分将被隐藏/裁剪。
    - **拉伸 (Fill)**：强行拉伸图片的物理比例以填满整个模块框，可能会造成画面变形。
- **[匹配媒体比例]**：勾选后，上述所有预设和宽高输入将被禁用，模块框的形状将直接由 ComfyUI 生成回传的真实图像/视频比例决定。

## 4. 媒体控制与资产管理 (右键菜单)

ShellLink 将模块内的媒体记录视为指向本地硬盘文件的**“软链接”**。理解“移除记录”与“删除文件”的区别，是高效管理资产的关键。

### 4.1 右键菜单核心功能

在任意模块（尤其是输出模块）上 **点击右键**，将呼出功能菜单：

- **内容操作 (针对媒体文件)**：
    - **下载 / 下载全部**：将当前媒体或该模块内的所有历史记录打包下载到本地。
    - **移除 (仅限此模块)**：**安全操作**。仅在当前模块的记录列表中删掉该图像/视频，不影响电脑本地硬盘里的实际文件，也不影响面板中其他克隆了该文件的模块。
    - **移除并删除本地文件**：**危险操作**。从整个面板的所有模块中拔除与该文件相关的所有记录（软链接），并**彻底从电脑硬盘（output 目录）中将其物理删除**。
    - **清除所有 / 清除所有并删除**：对当前模块内的所有历史记录执行上述操作。
- **模块操作**：
    - **选择相同模块** / **删除相同模块**。
    - **批量向前/向后移动**。

### 4.2 网格管理视图 (历史记录管理)

在动态工具栏点击“网格”图标，可进入生成记录管理模式。

- **多选交互**：按住 `Ctrl` (或 `Cmd`) 点击缩略图可进行多选，按住 `Shift` 可以进行范围连选。被选中的缩略图会亮起蓝框。
- **拖拽排序**：按住某张缩略图左右拖拽，可以自由改变其在历史记录中的排序位置。
- **快速清理**：
    - 点击缩略图右上角的 `✖` 等同于“安全移除（不删文件）”。
    - 选中多张缩略图后，可点击顶部的 **[移除]** 或 **[移除并删除文件]** 按钮进行批量操作。逻辑与右键菜单完全一致。

### 4.3 播放器与快捷键交互

ShellLink 内置了高度定制的沉浸式媒体播放器引擎：

- **历史记录无缝切换**：选中一个输出模块后，直接按下键盘的 **`←` (左)** 和 **`→` (右)** 方向键，即可如幻灯片般丝滑切换上一张/下一张图片或视频。
- **视频专属控制**：
    - 鼠标悬停在视频上，可唤出底部控制栏。
    - **调节音量**：点击或拖动右下角的音量滑杆，点击喇叭图标可一键静音。
    - **播放/暂停**：鼠标左键单击视频画面。
    - **拖拽进度**：在视频画面上按住 **鼠标右键** 左右拖动，可实现极其顺滑的视频帧拖拽预览；或者直接拖动底部的进度条。
    - **倍速与全屏**：点击右下角的 `⋮` 菜单，可调节播放速度 (0.5x - 5.0x) 或进入全屏/画中画模式。
- **音频专属控制**：包含黑胶唱片风格的 UI，支持独立进度条、波形模拟、音量滑杆与倍速胶囊按钮。
